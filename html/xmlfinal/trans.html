Transcriptions

The transcriptions are designed for use with the Mozilla Firefox browser. They display much less well if Microsoft Internet Explorer is used.



Master Transcription

A complete transcription of Harley MS 4431 which should be seen as a draft which is corrected and updated as errors and inconsistencies come to light. The current version dates from 27 January 2010.

master.xml
The transcription in XML. The file is extremely large.
