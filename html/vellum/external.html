Manuscript Viewer - Demo

This webpage displays the Virtual Vellum (VV) viewing environment within an external window to the web browser. The catalogue of available manuscripts is displayed in the top-left region of the VV window. Clicking on the name of a manuscript displays the available images as thumbnails. An image can be viewed in a main viewing window by either double-clicking the thumbnail or select the thumbnail once using the mouse and then clicking the "Open Thumbnail" button which is located to the left of the thumbnail preview scrollbar.
