An Index Proper Names used by Christine de Pizan in Harley 4431

We are working on a method to highlight Proper Names within the body text of our transcribed corpus. We hope then to provide some method of cross-reference into a list or index we maintain, and a further link or links to encyclopedia on the worldwide web. We have tagged about 5 000 entries in our transcription.

Following suggestions at the last 2 meetings, we have used Web 2.0 technologies to set up a wiki area that may only be edited by use of a password. This means we can write entries for each name in Harley 4431 and be assured of the url or web address to link to. For example the fixed reference to point to Queen Isabeau is:

https://www.wiki.ed.ac.uk/display/4431wiki/Queen+Isabeau

The Home page for our Index of Proper Names (L'Index des noms propres) is fixed at:
https://www.wiki.ed.ac.uk/display/4431wiki

When you create Proper Names please use the following guidelines:
  • No punctuation
  • No brackets
  • UK English spelling
  • No accents
Instructions for XML Tagging Names
Our XML tags then must use the new fixed reference. Please add the ref when you are transcribing, like this:

<name ref="Christine+de+Pizan">|crist|ine</name>

That ref Christine+de+Pizan may be obtained from our Index of Proper Names online at all times. Please copy and paste the ref from the web address starting just after the last slash / as in the screen-shot below:
